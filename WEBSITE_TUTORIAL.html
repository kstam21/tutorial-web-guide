üîç Part 2: Get Your Website on Google

6
Submit to Google Search Console

This is THE MOST IMPORTANT step! Without this, Google won't find your site.

1. Go to Google Search Console

Visit: https://search.google.com/search-console

Open Search Console ‚Üí
2. Sign In with Google Account

Use your Gmail or any Google account

3. Click "Add Property"

Usually a button in the top-left area

4. Choose "URL prefix"

NOT "Domain" - select the URL prefix option

5. Enter Your Website URL
https://username.github.io/repo-name/

Important: Include the trailing slash (/) at the end!

6. Click "Continue"

Google will now show verification options

7
Verify Ownership

Prove to Google that you own this website.

Method 1: HTML File Upload (Easiest)

Step 1: Google gives you a file (e.g., google1234abcd.html)
Step 2: Download this file
Step 3: Upload it to your GitHub repository
Step 4: Commit changes
Step 5: Wait 1 minute
Step 6: Click "Verify" in Search Console

Method 2: HTML Tag (Alternative)

Google gives you a meta tag. Add it to your index.html in the <head> section, then click "Verify"

‚úÖ Verification Successful!

You should see: "Ownership verified"

8
Submit Sitemap

Tell Google about all the pages on your website.

1. In Search Console, Click "Sitemaps"

Found in the left sidebar menu

2. Enter "sitemap.xml"

In the box, type: sitemap.xml

3. Click "Submit"

Status should change to "Success"

9
Request Indexing

Ask Google to crawl your site immediately.

1. Click "URL Inspection"

In the left menu at the top

2. Enter Your Homepage URL

Paste your website URL and press Enter

3. Click "REQUEST INDEXING"

This tells Google to crawl your site right away

4. Wait for Confirmation

You'll see: "Indexing requested" ‚úÖ

üéØ Done! Google Will Now Index Your Site

üíª Part 4: Understanding Your Website Files

HTML (Structure)

HTML is the skeleton of your website. It defines what content appears and where.

üí° What is HTML?

HTML (HyperText Markup Language) is like the blueprint of a house:

  • Headings (like room labels)
  • Paragraphs (like furniture descriptions)
  • Images (like pictures on the wall)
  • Links (like doors to other rooms)

Example HTML Code:

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Welcome!</h1> <p>This is a paragraph.</p> <img src="photo.jpg" alt="My Photo"> <a href="about.html">About Me</a> </body> </html>

Common HTML Tags:

  • <h1> = Main heading
  • <p> = Paragraph
  • <img> = Image
  • <a> = Link
  • <div> = Container
CSS (Styling)

CSS makes your website look beautiful. It's like decorating and painting your house.

üí° What is CSS?

CSS (Cascading Style Sheets) controls the appearance:

  • Colors (like paint)
  • Fonts (like text style)
  • Spacing (like room layout)
  • Animations (like moving decorations)

Example CSS Code:

/* Make headings blue */ h1 { color: blue; font-size: 32px; } /* Style paragraphs */ p { color: gray; line-height: 1.6; } /* Style buttons */ .button { background-color: purple; color: white; padding: 10px 20px; border-radius: 5px; }

How to Link CSS to HTML:

<link rel="stylesheet" href="style.css">
JavaScript (Interactivity)

JavaScript makes your website interactive and dynamic.

üí° What is JavaScript?

JavaScript makes things happen:

  • Buttons that respond to clicks
  • Forms that validate input
  • Menus that open and close
  • Animations and effects

Example JavaScript Code:

// Show alert on button click function showAlert() { alert("Button was clicked!"); } // Change color on hover document.querySelector('h1').addEventListener('mouseenter', function() { this.style.color = 'red'; });

How They Work Together:

  • HTML = Structure (the skeleton)
  • CSS = Presentation (the skin)
  • JavaScript = Behavior (the actions)

üÜò Troubleshooting Common Problems

Problem: Website Shows 404 Error

Cause: Files are in wrong location

Solution:

1. Check files are in repository root (not in a folder)
2. Make sure it's named exactly: index.html
3. Verify GitHub Pages is enabled

Problem: CSS Not Working

Cause: CSS file path is incorrect

Solution:

1. Check the CSS link in HTML: <link rel="stylesheet" href="style.css">
2. Make sure style.css is in the same folder as index.html
3. File names are case-sensitive

Problem: Not Showing on Google After 2 Weeks

Cause: Various possible reasons

Solution:

1. Check Google Search Console for errors
2. Verify robots.txt allows crawling
3. Make sure sitemap was submitted
4. Create more backlinks
5. Request indexing again

Problem: Images Not Loading

Cause: Image path is incorrect

Solution:

1. Check image src: <img src="photo.jpg">
2. If in folder: <img src="images/photo.jpg">
3. Make sure image is uploaded to GitHub
4. File names are case-sensitive
